# UI Icons: Killstreaks

This document contains AI image prompts and technical specifications for all 5 killstreak reward icons used in Zero Dawn: Modern Warfare. Each icon represents a killstreak reward displayed in the killstreak selection menu, HUD notification popup, and the killstreak progress tracker.

---

## Icon 1: UAV

### AI Image Prompt
A flat military-style icon depicting a simplified drone/UAV silhouette in top-down view with four rotor arms and a central sensor turret. The drone is rendered in pale grey-blue (#90CAF9) against a dark transparent background. Four concentric semicircular radar wave arcs emanate from the center, rendered in white (#FFFFFF) with decreasing opacity — symbolizing radar/recon functionality. The icon uses flat vector art style with a 2px white stroke outline for HUD visibility. The composition is centered and symmetrical. Colors: pale grey-blue (#90CAF9), white radar arcs (#FFFFFF), dark backdrop default. Inspired by UAV killstreak reward iconography from modern military shooters.

### Spec
- Format: PNG with transparency
- Resolution: 256x256
- Style: Flat vector, radar/recon theme
- Color palette: Pale grey-blue, white radar arcs
- Use case: Killstreak selection menu, HUD notification, streak progress bar
- Scalable: Yes — vector-compatible composition

## Icon 5: Nuke

### AI Image Prompt
A flat high-impact icon depicting a stylized nuclear explosion cloud — a classic mushroom cloud shape with a wide, churning stem and a flat, spreading cap. The cloud stem is rendered in orange-red (#FF5722) and the cap in dark grey (#424242) with a bright white (#FFFFFF) flash outline around the entire shape. A small radioactive trefoil hazard symbol (three-bladed design) is centered on the mushroom cap, rendered in yellow (#FFD700). The icon uses flat vector art with heavy contrast. The composition is centered and vertically oriented. Colors: orange-red stem (#FF5722), dark grey cap (#424242), yellow trefoil (#FFD700), white flash outline (#FFFFFF). Inspired by tactical nuke killstreak icons from Call of Duty franchise, adapted for original use.

### Spec
- Format: PNG with transparency
- Resolution: 256x256
- Style: Flat vector, high-impact/explosive theme
- Color palette: Orange-red, dark grey, yellow hazard symbol, white flash
- Use case: Killstreak selection menu, HUD notification, nuke warning overlay
- Scalable: Yes
