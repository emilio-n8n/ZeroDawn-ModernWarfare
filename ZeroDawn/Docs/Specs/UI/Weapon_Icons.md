This document contains AI image prompts and technical specifications for all 8 weapon icons used in Zero Dawn: Modern Warfare. Each icon represents a weapon displayed in the HUD weapon name/ammo display area, kill feed, Create-a-Class menu, and scoreboard.

## Icon 1: M4A1

### AI Image Prompt
A flat minimalist icon depicting an M4A1 assault rifle in side profile facing right, rendered in dark charcoal grey (#37474F) with a 2px white stroke outline for HUD readability. The weapon silhouette shows the distinctive features: a 14.5-inch barrel, triangular front sight post, M-LOK handguard, collapsible carbine stock, and 30-round STANAG magazine. The icon uses a simplified flat vector style that preserves recognizable weapon shape at small sizes (256x256). The composition is centered horizontally with the weapon oriented right. Colors: dark charcoal grey (#37474F), white outline (#FFFFFF). Inspired by weapon icon design in Call of Duty and Battlefield HUDs.

### Spec
- Format: PNG with transparency
- Resolution: 256x256
- Style: Flat vector silhouette, weapon profile
- Color palette: Dark charcoal grey, white stroke outline
- Use case: HUD weapon name area, kill feed, Create-a-Class menu, scoreboard
- Scalable: Yes — simplified silhouette works at multiple resolutions
