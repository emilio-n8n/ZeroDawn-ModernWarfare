# UI Icons: Perks

This document contains AI image prompts and technical specifications for all 10 perk icons used in Zero Dawn: Modern Warfare. Each icon represents a gameplay perk slot item displayed in the Create-a-Class menu, HUD loadout display, and in-game perk activation notifications.

---

## Icon 1: Flak Jacket

### AI Image Prompt
A minimalist military-style icon representing explosive resistance. The icon features a stylized fragmentation grenade silhouette in dark grey with an orange-yellow blast wave radiating outward, crossed out by a thick red diagonal prohibition line overlay. The composition is centered and symmetrical, using a flat vector art style with subtle 2px white stroke outlines. The background is transparent. Colors: dark grey (#3A3A3A), orange-yellow (#FF8C00), prohibition red (#E53935). Inspired by Call of Duty perk icon design language — simplified, flat, single-symbol clarity.

### Spec
- Format: PNG with transparency
- Resolution: 256x256
- Style: Flat vector, military minimalist
- Color palette: Dark greys, orange for explosive theme, red prohibition accent
- Use case: Create-a-Class perk slot, HUD perk activation indicator
- Scalable: Yes — designed as SVG-compatible flat vector
