## INSTRUCTIONS MESHY

1. **Text to 3D:** Colle le prompt sur https://www.meshy.ai → Text to 3D
2. **Image to 3D:** Upload l'image de référence + ajoute le prompt
3. **Réglages:**
   - Style: Realistic / PBR
   - Poly count: Medium (game-ready: 5k-15k tris)
   - Format: FBX ou GLB
4. **Après génération:**
   - Télécharge le FBX/GLB
   - Importe dans UE5 via Content Browser → Import
   - Place dans `/Game/ZeroDawn/Meshes/`
5. **Pour les armes:**
   - Oriente le barrel vers +X dans ton logiciel de modélisation
   - Scale: 1 unité UE5 = 1 cm
